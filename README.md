Sales Performance Dashboard
## Project Overview
This project is part of a Data Analyst Internship Task.
The objective is to design an interactive dashboard to analyze business sales performance.
## Tools Used
- Power BI
- Dataset: Superstore Sales Dataset
  ## Features
- KPI Cards (Total Sales, Total Profit, Total Orders)
- Sales Trend Analysis (Line Chart)
- Category-wise Sales (Bar Chart & Pie Chart)
- Region-wise Sales (Bar Chart)
- Interactive Filters (Slicer)
  ## Key Insights
- Sales show a consistent increasing trend over the years
- Technology category has the highest sales
- West region generates the highest sales
- Profit is lower compared to total sales
  ## Conclusion
This dashboard helps in understanding business performance and supports better decision-making.
